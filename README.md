# eslint、prettier、markdownのチュートリアル

## ESLint
**コードを解析し、ESLintで作ったルールに沿ったコーディングになってるかチェックするツール**  
- バグの温床になりそうなコードがないかチェックするツール
- コードの品質を担保するために使う　　

## Prettier
**コード自動整形ツール。コードに統一感を持たせるツール**
- インデントの付け方
- 折り返しルール等

## 重複ルールの取り扱い注意点
